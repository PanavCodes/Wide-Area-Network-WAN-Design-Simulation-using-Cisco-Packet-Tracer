### üß± Network Architecture
The network connects:
- **Headquarters (HQ):** Central hub with core routers and servers  
- **Branch Office 1, 2, 3:** Remote LANs connected through serial links  
- **Redundant paths** between HQ and branches for fault tolerance  
- **Partial mesh topology** for alternative routing  

Each site includes:
- Routers, Switches, PCs, and Servers (Mail, DNS, FTP, DHCP, NTP)
- RIP for route exchange and automatic topology learning
- ACLs for restricting unauthorized network access
